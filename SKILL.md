## Step 3 · 调抖音 API

```bash
curl -s --connect-timeout 10 --max-time 30 "{douyin.base_url}/dy_fetch_one_video_v3?aweme_id={ID}&apikey={douyin.apikey}"
```

或

```bash
curl -s --connect-timeout 10 --max-time 30 "{douyin.base_url}/dy_fetch_video_by_share_url?share_url={URL}&apikey={douyin.apikey}"
```

> {douyin.base_url} 和 {douyin.apikey} 替换为 config.json 中的实际值

**错误处理：**

| 情况 | 回复 |
|------|------|
| 网络/超时 | 「网络出了点问题，正在重试...」自动重试1次 |
| 首次失败 | 「第一次查询卡了一下，重试中... 不行的话检查 APIKey 是否激活 → rockmoons.com」 |
| 插件不存在 | 「功能还在准备中，稍后试试」 |

## Step 5 · 提取文案（ASR）

读 `references/asr-local.md`。快速流程：

1. 取数据中「视频音频」URL
2. 下载：`curl -o dy_audio_{id}.mp3 "{音频URL}"`
3. 首次检查环境：
   - `pip show faster-whisper || pip install faster-whisper`  
   - `pip show imageio-ffmpeg || pip install imageio-ffmpeg`
   - 首次下载模型：「第一次下载语音引擎（约1.5GB），像装App，以后就快了」
   - 设镜像：`export HF_ENDPOINT=https://hf-mirror.com`
4. 执行识别：
```bash
python3 -c "
from faster_whisper import WhisperModel
compute = 'int8' if '{asr.local.device}' == 'cpu' else 'float16'
model = WhisperModel('{asr.local.model_size}', device='{asr.local.device}', compute_type=compute)
segments, info = model.transcribe('dy_audio_{id}.mp3', language='zh')
print(' '.join([s.text for s in segments]))
"
```
5. 结果存为 `文案内容`，进度：`🎙️ 正在听写 ▓▓▓▓░░ 60% 约40秒`
6. 清理临时文件
7. 展示：`📝 文案内容（X分Y秒）：{文字}`
8. 操作提示：`💡 生成数字人视频 ✨ | 改写`

## Step 10 · 数字人口播视频

### 前置检查

**缺文案 →** 检查 session.last_script / 最近提取/改写的结果 → 复用 → 都没有 → 「请提供文案（直接写/发抖音链接提取/用之前提取的）」

**缺照片 →** 检查 defaults.avatar → 有则用 → 无则「还没设置形象，📷 上传照片 / 👩‍💼 用内置的」

**缺音色 →** 用 defaults.voice，没有就用系统默认

**首次用 →** 「你还没设置形象和声音。先用默认的试一个？🎬 试一个 / 🎨 先去设置」

### 模式判断

- config.heygem.mode = "cloud" → 调云端 API
- config.heygem.mode = "local" → 检查本地服务 → 调 127.0.0.1:8383
- config.heygem.mode = "" → 「先配置数字人：☁️云端 / 💻本地？」

### 字幕

默认开启（defaults.subtitle = true）。用户说「不要字幕」则跳过。

生成完视频 → ffmpeg 烧字幕 + 生成 .srt：
```bash
ffmpeg -i video.mp4 -vf "subtitles=sub.srt:force_style='FontSize=24,PrimaryColour=&H00FFFFFF,OutlineColour=&H00000000'" output.mp4
```

### 生成 & 验证

本地模式：
```bash
curl -X POST http://127.0.0.1:8383/easy/submit -F "image_path={图片}" -F "text={文案}" -F "voice={音色}"
```
等待 → 下载/获取视频

云端模式：
```bash
curl -X POST "{api_url}/submit" -F "image=@{图片}" -F "text={文案}" -F "voice={音色}"
```
轮询进度，完成后下载

**验证：** `✅ 画面 OK ✅ 声音 OK ✅ 字幕已嵌入`

**进度：** `🎬 让数字人开口说话 ▓▓░░░░ 35% 约3分钟 你可以先做别的～`
