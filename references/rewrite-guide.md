# 文案改写/二创 Prompt 指南

触发：用户说「改写」「去重」「换个说法」「缩写」「扩写」「换风格」「提取金句」「拆条」等。

改写不需要外部 API，由模型直接完成。

---

## 交互规则

### 模糊请求
如果用户只说「改写」而没有指定模式：
```
模型问：「要哪种改写？
  1. 去重改写 — 换表达不换意思
  2. 缩写 — 浓缩到 30 秒 / 15 秒
  3. 扩写 — 补充细节场景
  4. 换风格 — 小红书 / 新闻稿 / 幽默
  5. 提取金句 — 抓最精彩的一两句
  6. 拆条 — 一段拆成多条」
```

### 明确请求
用户一句话说全（如「改成小红书风格」）→ 直接执行，不追问。

---

## 6 种模式 Prompt 模板

### 1. 去重改写
```
你是一个文案改写助手。请对以下抖音视频文案进行去重改写：
- 保持原意不变
- 换用不同的表达方式、句式、词汇
- 避免和原文结构雷同
- 长度与原文字数相近
- 保持口语化、接地气的风格

原文：
{文案内容}

改写：
```

### 2. 缩写
```
你是一个文案精简助手。请将以下抖音视频文案缩写成约 {时长} 的版本：
- 保留核心信息和最精彩的表达
- 删除冗余、铺垫、重复内容
- 保持口语化
- 目标时长：{时长}

原文（{原时长}）：
{文案内容}

缩写版：
```

### 3. 扩写
```
你是一个文案创作助手。请对以下抖音视频文案进行扩写：
- 补充更多细节、场景描写、情绪渲染
- 增加铺垫和转折，让叙事更丰满
- 保持原风格和口吻
- 长度扩展约 1.5-2 倍

原文：
{文案内容}

扩写版：
```

### 4. 换风格
```
你是一个风格转换助手。请将以下抖音视频文案改写成 {目标风格} 风格：
{风格说明}

原文：
{文案内容}

{目标风格}版：
```

**风格说明字典：**

| 风格 | 说明 |
|------|------|
| 小红书 | 用「🔥」「✨」等 emoji，短句多换行，感叹号，种草语气，加话题标签 |
| 新闻稿 | 客观冷静，第三人称，规范书面语，5W1H 结构 |
| 幽默 | 加梗、反转、自嘲、夸张比喻，口语化 |
| 严肃/深度 | 长句、逻辑严密、用学术词汇，减少口语 |
| 鸡汤/励志 | 正能量、排比句、比喻、温暖治愈 |
| 悬疑 | 设问、留白、层层递进，制造悬念 |

### 5. 提取金句
```
你是一个内容提炼助手。请从以下抖音视频文案中提取 1-3 句最精彩、最有冲击力的句子：

要求：
- 能单独作为标题或封面文案
- 有情绪张力或信息密度
- 不超过 50 字/句
- 优先提取反转、高潮、点睛之笔

原文：
{文案内容}

金句：
```

### 6. 拆条
```
你是一个内容拆分助手。请将以下抖音视频文案拆成 {N} 条独立的短视频文案：

要求：
- 每条有独立主题，可单独成片
- 每条 30-60 秒的语速长度
- 每条有吸引人的开头
- 风格一致

原文：
{文案内容}
```

---

## 输出格式

```
📝 原文（{X分Y秒}）
{原文全文}

🔄 改写 · {模式名称}
{改写结果}
```

---

## 数据关联

改写完成后，追加以下字段到视频数据：
- `改写文案` = 改写结果文本
- `改写模式` = 使用的模式名称（如「小红书风格」「去重」「缩写30秒」）

导出的 Excel 中会多出这两列。
