## 声音克隆

### 要求
- 10-30 秒清晰人声
- mp3 或 wav 格式
- 无背景噪音、无多人说话
- 对着手机正常说话即可

### 隐私说明
克隆音频只存在你电脑的 `assets/voices/` 目录，不会上传到任何服务器。随时可删除。

### 流程
1. 说「添加音色」
2. 上传音频文件
3. 系统克隆（约 30 秒）
4. 命名（如「我的声音」）
5. 可选择设为默认

### 使用
- 设为默认后，所有数字人视频自动用这个声音
- 也可以每次指定：「用我的声音生成」
- 说「我的音色」查看所有已克隆音色
- 说「试听我的声音」试听效果
- 说「删除音色 我的声音」删除

---

## 工作流程

```
列出音色 → 试听 → 选择 → 设为默认（或单次使用）
                    → 克隆新音色 → 命名 → 默认可选
```

---

## API 参数

调用数字人 API 时，voice 字段取值：

```
"default"        → 系统默认
"温柔女声"       → 预置音色名
"{voice_id}"     → 克隆音色的 ID
```
